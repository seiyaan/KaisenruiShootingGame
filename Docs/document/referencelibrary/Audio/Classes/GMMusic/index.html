Instant Methods

GMMusic

GMMusic(const std::string& filename, bool loop=true)
ファイル名（拡張子含む）を指定して、BGM再生用のインスタンスを作成します。

Discussion

デフォルトではループ再生しますが、第2引数に false を指定するとループ再生しないようになります。

play

void play()
BGM の再生を開始します。

pause

void pause()
BGM の再生を一時停止します。

stop

void stop()
BGM の再生を中断します。

Discussion

このメソッドの呼び出し後、BGM の再生位置は先頭に戻ります。

isPlaying

bool isPlaying() const
BGM が再生中かどうかを取得します。

getVolume

BGM の音量を設定します。
