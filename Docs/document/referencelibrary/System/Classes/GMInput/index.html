GMInput Class

Super Class

  • GMObject

Overview

キーボードとマウスの入力を管理するためのクラスです。

このクラスのインスタンスには、クラス変数 GMInput::Inst を使ってアクセスできます。

getTouchCount

getTouchLocationAny

GMVector2D getTouchLocationAny() const
現在タッチされている画面上の位置をいずれか1点取得します。

Discussion

1点もタッチされていない場合には、X座標、Y座標ともに DBL_MAX がセットされた GMVector2D がリターンされます。

getTapCount
