Karakuri 2 API リファレンス

Karakuri 2 は、純粋な C++ で記述できる、2D・3D ゲーム開発フレームワークです。

Karakuri の歴史についてはこちら。

Karakuri 2 について

フレームワークについて

  • フレームワークで用意しているクラス・構造体・列挙型・グローバル関数には、「GM」というプリフィクスがついています。
  • クラスメソッドは大文字から、インスタンスメソッドは小文字から始まります。
  • 名前が「__」から始まるクラス・構造体・列挙型・関数は内部での使用を想定していますので、使用しないでください。

ゲーム全体について

  • ゲームの基本設定は、Globals.h と Globals.cpp の値で変更できます。
  • Mac OS X / iOS などのターゲットによって、利用できる機能が少しずつ異なります。
  • シーンの変更は、GMGame::ChangeScene("シーン名") で行います。

描画について

  • フォント一覧はこちらです。
  • カラーチャートはこちらです。
  • 描画は基本的に左下を原点として行われます。
  • テクスチャ座標も左下を原点とした座標で (0, 0)〜(1, 1) の値で指定します。

下図に、Karakuri 2 の描画関係のクラスの概略を示します（クリックで大きいバージョンを表示）。

オーディオについて

  • WAV, AIFF, MP3, AAC といった一般的なオーディオファイルをサポートしています。
  • iOS デバイス上での使用には、こちらを参照して、CAF 形式に変換しておくことを推奨します。
  • すべての環境で保証されている同時再生数は、BGM 1ソース、SE 1ソースです。

乱数について

  • 乱数は、GMRandom クラスのクラスメソッドを利用してください。
