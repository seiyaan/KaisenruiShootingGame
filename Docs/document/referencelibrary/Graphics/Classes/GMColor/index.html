Instant Methods

GMColor

GMColor()
赤、緑、青の各色成分が 0 で、アルファ成分が 255 の色を作成します。

GMColor

GMColor(double r, double g, double b, double a)
赤、緑、青、アルファの各色成分を 0.0〜1.0 で指定して色を作成します。

GMColor

GMColor(const GMColor& color)
渡された色と同じ色情報をもつ色を作成します。

GMColor

GMColor(const GMVector3D& vector)
渡されたベクトルを利用して、r=x, g=y, b=z となるように、アルファ成分が 255 の色を作成します。

GMColor

GMColor(const GMVector4D& vector)
渡されたベクトルを利用して、r=x, g=y, b=z, a=w となるように色を作成します。
