GMLabel Class

Super Class

  • GMObject

Overview

テキストラベルを表すためのクラスです。

このクラスは、フォントの名前と大きさを指定して作成し、表示したいテキストを適宜セットします。
描画が必要なときには、getTexture() メソッドでテクスチャを取得して描画してください。
テクスチャは、白のテキスト色で用意されます。

利用できるフォントの名前は、開発ガイドの「フォント一覧」を参照してください。
