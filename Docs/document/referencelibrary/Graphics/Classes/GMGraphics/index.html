GMGraphics Class

Super Class

  • GMObject

Overview

ビューの大きさ、ポイントスプライトの設定、デプステストの設定などを管理するためのクラスです。

このクラスのインスタンスには、クラス変数 GMGraphics::Graphics1 または GMGraphics::Graphics2 を使ってアクセスできます。
