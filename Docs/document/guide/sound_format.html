サウンド形式について

利用可能なオーディオファイルの形式

  • オーディオファイルには、22.05kHz あるいは 44.1kHz の CAF (Core Audio File) 形式 (〜.caf) の利用を推奨します。
  • CAF ファイルの内部エンコーディングは、BGM であれば AAC、効果音であれば IMA4 を推奨します。
  • WAV、AIFF、MP3、AAC などの一般的な形式のオーディオファイルも利用できます。
  • MP3 よりは AAC の方が同じサイズにおいて音質が良くなります。

iOS SDK でのオーディオファイルの変換方法

iOS SDK がインストールされている環境下で、ターミナルから以下のように afconvert コマンドを実行します。

BGM 用のファイル（ステレオ）
% afconvert -f caff -d aac@44100 -c 2 [元ファイル] [出力ファイル.caf]
効果音用のファイル（モノラル）
% afconvert -f caff -d ima4@22050 -c 1 [元ファイル] [出力ファイル.caf]
効果音用のファイル（ステレオ）
% afconvert -f caff -d ima4@22050 -c 2 [元ファイル] [出力ファイル.caf]
