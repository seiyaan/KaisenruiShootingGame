その他の FAQ

FPS の表示はどうやったら消せますか？
GameMain.cpp の中で、setShowsFPS() 関数の呼び出しの引数を false にしてください。
Xcode のオーガナイザで「A valid signing identity matching this profile could not be found in your keychain」というエラーが出て、実機用のビルドができません。
この問題は、開発用プロファイルを作成したマシンと、現在開発を行っているマシンが異なる場合によく起こります。開発中のマシンに、使用しているプロファイルに対応した証明書の秘密鍵もコピーしていることを確認してください。
iPhone を縦モードで使うにはどうすればいいですか？
GameMain.cpp で、デフォルトで横モード用に設定されている画面サイズ (480, 320) を、(320, 480) にして使ってください。
