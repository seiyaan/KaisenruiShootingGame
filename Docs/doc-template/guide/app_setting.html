アプリケーションの設定

アプリケーション名の設定

「Mac OS X」ターゲットあるいは「iPhone」ターゲットをダブルクリックして、「ビルド」設定で「プロダクト名」を変更してください。

識別子の設定

iPhone 実機で動作させるためには、使用するプロファイルに合致する識別子を設定しなければなりません。「iPhone」ターゲットをダブルクリックして、「プロパティ」設定で「識別子」を変更してください。

タイトルの設定（Mac OS X のみ）

GameMain クラスのコンストラクタで、setTitle() 関数の引数を変更してください。

アイコンの設定

Mac OS X 用のものは macosx_icon.icns を適切なアイコンファイルで置き換えてください。

iPhone 用のものは iphone_icon.png を適切な PNG 画像で置き換えてください。画像サイズは 57×57 ピクセルである必要があります。

起動画面の設定

Default.png を適切な画像で置き換えてください。この画像は、480×320 ピクセルの画像を時計回りに90度回転させた PNG 画像である必要があります。
