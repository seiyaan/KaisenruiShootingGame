フォントについて

はじめに

文字列を画面に描画するためには、KRFont クラスを使用します。このクラスはフォント名とサイズを指定して作成します。文字列のテクスチャを生成することができますが、直接文字列を描画するための関数はもっていません。

生成されるテクスチャの文字色は、常に白になります。KRTexture2D クラスに描画色を指定して描画できる関数が用意されていますので、文字色の指定はテクスチャの描画時に行ってください。
